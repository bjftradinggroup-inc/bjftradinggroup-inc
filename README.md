## BJF Trading Group Inc.

Forex execution research, broker dialect engineering, and arbitrage trading infrastructure.
Based in Ontario, Canada.

### Active open-source

**[BEQI — Broker Execution Quality Index](https://github.com/bjftradinggroup-inc/forex-broker-audit-toolkit)**
Open methodology and Python toolkit to measure retail forex broker execution quality across five dimensions: matching latency, slippage asymmetry, spread widening, last-look hold, requote rate. MIT licensed.
Methodology document: [bjftradinggroup.com/forex-broker-audit-toolkit](https://bjftradinggroup.com/forex-broker-audit-toolkit/)
